# RAG-E
**Retrieval Augmented Generations** for **Everyone**.
### Start API with Docker
To build Dockerfile in /api folder:
```bash
docker build . -f api/Dockerfile -t rage-api
```
To start Docker Container:
```bash
docker run -it --gpus=all rage-api  
```
